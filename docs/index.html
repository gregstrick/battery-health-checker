Manual QC checklist

Excel logging built in

First run

  • macOS: right-click the app → Open the first time (unsigned app, Gatekeeper will warn once).
  • Windows: click "More info" → "Run anyway" if SmartScreen warns (unsigned .exe).
  • First time you plug in each physical device, tap Trust on its screen and enter its passcode. After that, it connects instantly with no prompts.
  • Each computer keeps its own separate device log — they don't sync between Mac and Windows.
